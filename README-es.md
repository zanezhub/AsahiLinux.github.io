# Página principal

Este es el repositorio para la página principal de Asahi Linux.

Esta página está utilizando [Hugo](https://gohugo.io) y está siendo hosteada en Github pages en https://asahilinux.org.

## Primeros pasos

Si quieres contribuir, primero tienes que clonar este repositorio de manera recursiva ya que contiene un [submódulo](https://git-scm.com/book/en/v2/Git-Tools-Submodules).

```bash
git clone --recursive https://github.com/AsahiLinux/AsahiLinux.github.io
```

El siguiente paso es asegurarte de instalar Hugo. La mejor manera de instalar dependerá de la plataforma que estés usando, te recomendamos revisar su [documentación](https://gohugo.io/getting-started/installing/).

Una vez que todo esté instalado, entra al directorio en que este repositorio se clonó y ejecuta los siguientes comandos para empezar un server de desarrollo

```bash
cd AsahiLinux.github.io
hugo server
```

Ahora vas a entrar al servidor web que está siendo ejecutado de manera local en http://localhost:1313.
